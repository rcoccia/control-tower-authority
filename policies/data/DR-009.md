# DR-009: Confidential RelayEU data residency

> **SYNTHETIC DEMO POLICY** — This is synthetic content for a bounded public
> demonstration. It is not organizational policy or a production decision.

## Intent

Keep RelayEU confidential data classified, resident in the EU, and retained
only for its defined purpose.

## Applicability

Applies when `data_classification` equals `confidential` and `jurisdiction`
equals `EU`.

## Required properties

<a id="dr-009-confidential-classification"></a>
### `dr-009-confidential-classification` — Classification

Label and handle the data as Confidential.

<a id="dr-009-eu-residency"></a>
### `dr-009-eu-residency` — EU residency

Store and process the data within the EU.

<a id="dr-009-no-third-party-transfer"></a>
### `dr-009-no-third-party-transfer` — Third-party transfer

Do not transfer the data to third parties.

<a id="dr-009-retention"></a>
### `dr-009-retention` — Retention

Define and enforce a purpose-bound retention period.

## Expected evidence

Provide the data inventory, classification, regional storage and processing
design, transfer analysis, and retention/deletion configuration.

## Exceptions and waivers

This demo policy grants no exception or waiver. Record a request for owner
review; only the simulated policy-owner process may publish a future change.

## Progressive disclosure

Start with the property IDs above. See `classify-data` in `.github/skills/`
for fact elicitation; it is advisory only.
